以下の手順でビルドとテストを実行し、結果を報告してください。

## 手順

1. **ビルド確認**
   ```bash
   cd H:/Codex/DevNext && dotnet build DevNext.sln
   ```
   ビルドエラーがあれば内容を報告し、修正を提案する。

2. **テスト実行**
   ```bash
   cd H:/Codex/DevNext/Tests && dotnet test
   ```
   テストの合否・件数・失敗内容を報告する。

## 報告形式

- ビルド: 成功 / 失敗（エラー内容）
- テスト: X件合格 / X件失敗（失敗内容と原因）
- 問題があれば修正案を提示する
